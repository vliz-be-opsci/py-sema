# CLI Reference

py-sema exposes the following CLI commands:

- sema
- sema-bench
- sema-conneg
- sema-get
- sema-harvest
- sema-query
- sema-query-templates
- sema-subyt
- sema-syncfs
- sema-roc

For command-specific options, run:

```bash
<command> --help
```

Examples:

```bash
sema-query --help
sema-harvest --help
sema-bench --help
```
